Flamerock's Flash hub

The purpose of this site is to keep old Adobe Flash games playable without downloading software. If you are the copyright owners and want any of these games removed make a pull request on GitHub. This site uses Ruffle to emulate Flash it's still in early development and lots of games have issues. If you want to see a game added, add a pull request on GitHub.
