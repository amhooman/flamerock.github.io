Flamerock's Flash Collection

The purpose of this site is to keep old Adobe Flash games playable without downloading software. This site uses Ruffle to emulate Flash it's still in early development and lots of games have issues. If you want to see a game added, add an issue on GitHub.

This site doesn't have any ads or trackers any aims to be lightweight

If you are the copyright owners and want any of these games removed make an issue on GitHub.

So far here are the list of games:

  • Bloons Tower Defense
  • Bloons Tower Defense 2
